# PA-Workspace – Struktur

Dieser Ordner ist zugleich ein **Obsidian-Vault**: plain-Markdown-Notizen + zugehörige Dokumente (PDFs/Docs) liegen nebeneinander.

- **00_Inbox/** – Drop-Zone. Neue Scans/Downloads hier rein, Claude sortiert.
- **01_Vorgänge/** – aktive Fälle, je Fall 1 Ordner mit gleichnamiger Index-Notiz.
- **02_Reisen/** – je Trip 1 Ordner.
- **03_Stammdaten/** – Master-Daten & Dokumente.
- **04_Personen/** – Profile, Vorlieben, Geschenkideen.
- **05_Finanzen/** – Rechnungen, Belege, Amex.
- **09_Archiv/** – abgeschlossene Vorgänge (nach Jahr).

**Einstieg:** [[00_Dashboard]]

**Konvention:** Jeder Vorgang-Ordner enthält eine Index-Notiz mit gleichem Namen wie der Ordner. Status/To-dos/Fristen stehen im YAML-Frontmatter und im Text. Verlinkung über `[[Wikilinks]]`.
